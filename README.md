# Sentiment-Aware Customer Support Assistant

This project is being developed as part of an internship project under the guidance of Prof Abhishek Srivastava Sir.

The objective of this project is to fine-tune a small language model for customer support reply generation and compare different parameter-efficient fine-tuning techniques such as:

- Full Fine-Tuning
- LoRA
- QLoRA
- Prompt/Prefix Tuning

The project also includes a lightweight backend and frontend application for testing and comparing generated responses.

---

## Tech Stack

- Python
- Hugging Face Transformers
- PEFT (LoRA / QLoRA)
- Flask
- Streamlit
- Plotly
- Google Colab

---

## Features

- Customer support reply generation
- Fine-tuning comparison dashboard
- Side-by-side response comparison
- Evaluation metrics visualization
- Lightweight deployment
